#Funkciju definēšana

Python programmēšanas valodā funkcijas definē ar vārdu def.

~~~python
def FUNKCIJAS_NOSAUKUMS():
    print("Funkcijas darbība")
~~~

Piemērs.
~~~python
def hello():
    print("Sveika pasaule")
hello()
~~~
