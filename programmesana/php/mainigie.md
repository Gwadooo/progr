# Mainīgie

Visi mainīgie tiek definēti ar `$` zīmi. Mainīgā pieraksta beigās liek semikolu - `;`

~~~php

$skaitlis = 0;
$teksts = "Teksts";
$status = True;
$peldpunkts = 0.3;
$masivs = array();

~~~
